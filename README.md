■■■　スロット集計ツール　■■■　2025/12/06

【はじめに】
　本ツールはジャグラーのデータ取り込みを最適化し、
　日々の入力から集計・グラフ化までを自動化します。
　手作業を大幅に省き、効率的な分析を支援します。
　複数店舗のデータもまとめて管理できます。

【起動方法】
　同フォルダ内の「LauncherApp」フォルダにある
　「LauncherApp.exe（中継アプリ）」を起動してください。
　中継アプリが本体アプリを起動します。
　中継アプリのその他の役割は次のとおりです。
　　① デスクトップへショートカットを作成する
　　② タスクバーにアイコンを表示する
　　③ 起動時のセキュリティ警告を非表示にする

【サポート】
　続行が困難な問題が発生した場合は、
　「Link」フォルダ内の「問合せ先URL」よりご連絡ください。
